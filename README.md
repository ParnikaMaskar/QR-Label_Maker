# QR Sticker Generator

React + TypeScript app to generate bulk QR code stickers from CSV/Excel and export them as a printable PDF (A4, A3, or custom size).

CSV / Excel Format
Required columns:
model, serial, mac, host, ip

## Requirements
- Node.js (v16+ recommended)
- npm

## Install & Run

```bash
npm install
npm run dev
```
Open in browser:
http://localhost:5173
